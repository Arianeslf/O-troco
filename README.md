# 💰 Sistema de Caixa Eletrônico

Este projeto é um sistema simples de caixa eletrônico desenvolvido em **JavaScript**, com o objetivo de simular operações bancárias básicas diretamente no navegador.

---

## 🚀 Funcionalidades

* 🔎 **Consultar saldo**
* 💵 **Realizar depósitos**
* 💸 **Realizar saques**
* 📄 **Visualizar extrato**
* 🚪 **Encerrar o sistema**

---

## 🧠 Como funciona

Ao iniciar o sistema, o usuário informa seu nome e recebe uma mensagem de boas-vindas.

Em seguida, ele pode navegar por um menu interativo e escolher uma das opções disponíveis:

1. Saldo
2. Depósito
3. Saque
4. Extrato
5. Sair

---

## 📄 Extrato

O sistema registra automaticamente todas as movimentações realizadas, como:

* Depósitos
* Saques

Essas informações são armazenadas em um **array** e exibidas ao usuário em formato de lista.

---

## ⚠️ Validações

O sistema possui validações importantes para evitar erros:

* Não permite valores inválidos (letras ou vazio)
* Não permite saque maior que o saldo
* Não permite valores negativos ou iguais a zero
* Exibe mensagens de erro quando necessário
